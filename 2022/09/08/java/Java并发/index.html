AQS

设置一个CLH队列管理获取不到锁的线程

  1. 通过CAS对同步状态进行原子操作修改值
  2. 共享方式
    1. 独占：只有1个线程能执行
      1. ReentrantLock
        1. 可以实现公平锁和非公平锁
    2. 共享：可以允许多个线程执行
      1. Semaphore
        1. acquire和release进行锁的获取和释放
        2. 有非公平和公平模式
      2. CountDownLatch(倒计时器)
        1. 允许count个线程阻塞在一个地方，直到所有的线程都执行完毕
        2. 典型用法
          1. 某一个线程在开始运行前等待n个线程执行完毕
            1. 设置计数器为n
            2. 某个线程执行wait
            3. n个线程执行countDown方法
          2. 实现多个线程开始执行任务的最大并行性
            1. 设置计数器为1
            2. 多个线程执行wait
            3. 主线程执行countDown
            4. 多个线程并发
