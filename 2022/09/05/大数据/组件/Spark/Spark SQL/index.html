Spark SQL

Spark SQL

  1. 用SQL查询数据
  2. 以DataFrame或者DataSet的形式返回数据

加载数据

  1. 支持从多种文件加载数据
  2. 可以从Hive表中加载，并且用SQL对表进行查询

Join的方法

  1. broadcast hash Join
    1. 小表分发到每个节点，在本地join
    2. 使用hash值进行join
  2. shuffle hash join
    1. 两张表拆开，根据key分发到不同节点上join
    2. 使用hash值进行join
  3. sort merge join
    1. 两张表拆开，根据key分发到不同节点上
    2. 两张表的key预先排序
    3. 分别遍历两张表进行join

执行过程

  1. ANTLR得到抽象语法树
  2. 根据抽像语法树和元数据得到逻辑执行计划
  3. 逻辑执行计划优化
    1. 基于规则的优化
      1. 谓词下推
      2. 常量折叠
      3. 行列裁剪
    2. 基于代价的优化
      1. 逻辑计划生成多个物理计划，从中选择最小的
  4. 得到物理执行计划
  5. 执行物理执行计划
    1. 生成java字节码
    2. 划分DAG
    3. 生成RDD
