Python为什么没有重载？

对于Java和C++的使用者，使用重载实现同名函数的不同功能非常常见。

但是在Python中不支持这样的重载，因为Python中支持可变参数，可以通过*args和**kwargs传入不同长度的参数。

可以通过判断类型的方式让Python执行不同的逻辑。

如果需要实现和Java更接近的重载，可以使用functools.singledispatchmethod
