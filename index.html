git操作总结

git的操作

修改全局配置

已修改默认分支名为例：git config --global init.defaultBranch main

一般地，git的默认配置文件目录是C:\Users\your_user_name\.gitconfig

关联远程项目和分支

有时本地创建仓库后再在github上创建仓库，需要关联远程项目和分支

关联远程仓库

git remote add origin https://github.com/xxx/oa.git

关联本地分支和远程分支

git branch --set-upstream-to=origin/main main
